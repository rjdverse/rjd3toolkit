# Differencing of a series

Differencing of a series

## Usage

``` r
differences(data, lags = 1, mean = TRUE)
```

## Arguments

- data:

  The series to be differenced.

- lags:

  Lags of the differencing.

- mean:

  Apply a mean correction at the end of the differencing process.

## Value

The differenced series.

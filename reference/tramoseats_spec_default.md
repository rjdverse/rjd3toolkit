# Default Tramo-Seats specification

XTramo-Seats default specification generated with rjd3tramoseats used in
examples for specification customization functions in rjd3toolkit

## Usage

``` r
tramoseats_spec_default
```

## Format

An object of class `JD3_TRAMOSEATS_SPEC` of length 3.

## Examples

``` r
data(tramoseats_spec_default)
```
